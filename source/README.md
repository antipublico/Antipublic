## Editable source project (reconstructed)

This folder (`editable-source/`) is a clean React project intended for **editing site text in `src/`**.

Important:
- This was reconstructed from the production bundle’s embedded strings.
- It’s meant to be **easy to edit and re-host**, not a perfect byte-for-byte reproduction.

### Requirements
- Install **Node.js (LTS)** and npm

### Run locally

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
npm run preview
```

### Where to edit text
- `src/content.ts`: all the copy, links, projects, experience, etc.
- `src/App.tsx`: layout/structure
- `src/styles.css`: styling
